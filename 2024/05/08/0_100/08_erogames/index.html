# 操作说明

RPG 可自由操作时两指齐按（右鍵）打开功能表存档，部分需定点存档
IOS 音频已修正，但仍有部分游戏可能并无声音 修复 RPGMV 在 IOS 上无法运行 ogg 文件

无科学环境可能造成卡顿、画面加载不全等影响

游戏于 github page , 如有需要可从 git 仓库自行 Clone 本地

  • 夏日狂想曲
  • 冬日狂想曲
  • 彼女と痴漢と満員電車

  • 寻找夏日的宝物 生肉
