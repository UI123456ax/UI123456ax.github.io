# hexo-helper-live2d 插件安装与配置

hexo 的官方是支持看板娘的，已经封装好了插件

# 安装插件

在站点根目录下打开终端，输入以下命令

npm install --save hexo-helper-live2d

# 选择并安装模型

以 live2d-widget-model-shizuku 模型为例

npm install --save live2d-widget-model-shizuku

点击预览更多模型
